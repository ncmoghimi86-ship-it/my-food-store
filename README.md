# My Food Store  
**سفارش آنلاین غذا – ریسپانسیو**

Live Demo → 



## ویژگی‌های کلیدی
- منوی غذاها با دسته‌بندی وقابلیت سرچ  
- سبد خرید هوشمند (افزودن / حذف / محاسبه خودکار)  
- فرم سفارش کامل (نام، آدرس، انتخاب موقعیت روی نقشه، تاریخ و ساعت تحویل)  
- ذخیره سفارش‌ها با Redux Toolkit  
- داشبورد مینیمال  نمایش آخرین خرید  
- ورود / ثبت‌نام / خروج کاربر  
- کاملاً ریسپانسیو + منوی همبرگری در موبایل  
 

## تکنولوژی‌های استفاده شده
| تکنولوژی            | توضیحات                          |
|----------------------|---------------------------------|
| React 18 / 19        | آخرین نسخه                      |
| Redux + Redux Toolkit| مدیریت سراسری حالت              |
| React Router v6      | مسیریابی                        |
| Ant Design (v5)      | کامپوننت‌های زیبا و آماده        |
| Leaflet / React-Leaflet| نقشه تعاملی برای انتخاب آدرس  |
| Vite                   | باندلر سریع                   |
